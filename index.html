We multiplied below probability of getting same number 3,4,5 by 3 because roll is made up with 3 rows.

  • Probability of getting same number 3 times in a row is (1/5 * 1/5 * 1/5) * 3 = 3/125 = 0.024
  • Probability of getting same number 4 times in a row is (1/5 * 1/5 * 1/5 * 1/5) * 3 = 3/625 = 0.0048
  • Probability of getting same number 5 times in a row is (1/5 * 1/5 * 1/5 * 1/5 * 1/5) *3 = 3/3125 = 0.00096
